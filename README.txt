== Description ==

🚀 **No Facebook API Key Required!** Our intelligent scraping engine fetches events directly — no app creation, no tokens, no hassle.

== Installation ==

= Install from WordPress Dashboard =

1. Log in and navigate to **Plugins → Add New**.
2. Search for **"XT Widget for Facebook Events"**.
3. Click **"Install Now"** and then **"Activate"**.
4. Go to **Facebook Events → Live Feeds** to create your first event feed.

= Manual Installation =

1. Download the plugin from WordPress.org.
2. Unzip and upload the `xt-facebook-events` folder to `/wp-content/plugins/`.
3. Activate the plugin from the **Plugins** screen.
4. Navigate to **Facebook Events → Live Feeds** to get started.

= Quick Start =

1. Click **"Add New Feed"** from the Live Feeds page.
2. Choose your event source (Event IDs, iCal URL, or Page/Group ID with Pro).
3. Select a layout and configure display settings.
4. Publish the feed and copy the shortcode or use the Gutenberg block.

== Frequently Asked Questions ==

= Do I need a Facebook API key? =
No! The Live Feed Builder uses an intelligent scraping engine that works without any API keys or Facebook app setup. The legacy widget method still supports optional Facebook App authentication.

= What event sources are supported? =
The free version supports **Event IDs** and **iCal URL** sources. The [Pro version](https://xylusthemes.com/plugins/xt-facebook-events/) adds **Facebook Page ID** and **Facebook Group ID** sources for automatic event fetching.

= Can I display events from a Facebook Group? =
Yes, but this requires the [Pro add-on](https://xylusthemes.com/plugins/xt-facebook-events/). With Pro, simply paste your Group URL or ID and the plugin will automatically fetch all upcoming events.

= How many layouts are available? =
The free version includes **Card Grid** and **List View**. The [Pro version](https://xylusthemes.com/plugins/xt-facebook-events/) adds **Timeline**, **Compact List**, **Minimal Grid**, **Ticket List**, and **Masonry** layouts — 7 layouts total.

= Will events update automatically? =
Yes! Events are cached for the duration you configure (default: 24 hours). When the cache expires, fresh events are fetched automatically. You can also manually clear the cache from the feed settings.

= Does it work with Gutenberg? =
Absolutely! Use the **XT Facebook Events Feed** block to embed any feed directly in the block editor with a simple dropdown selector.

= Can I hide online/virtual events? =
Yes. Each feed has a "Hide Online Events" toggle in the Filters tab that automatically detects and hides virtual events (including multi-language detection).

= Is the plugin GDPR compliant? =
The plugin fetches public event data from Facebook and caches it on your server. No user tracking or third-party cookies are used. Event data is displayed directly from your WordPress database cache.
